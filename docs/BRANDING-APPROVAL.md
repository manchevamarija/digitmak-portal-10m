# DigitMak Portal — branding candidate and approval record

## Implemented candidate

The V11 public portal styling was aligned on 21 July 2026 with the publicly visible visual language at [digitmak.mk](https://digitmak.mk/). It uses the existing DigitMak SVG logo, the public-site red `#D11525`, yellow `#FFCC29`, near-black `#14171C`, Inter body text, Fraunces display headings, compact six-pixel action corners and a light grid/network hero treatment.

The hero content is available in Macedonian, English and Albanian. It mirrors the public project's visual direction and highlights the AI help desk, test-before-invest support and the one-stop-shop portal without copying WordPress implementation code.

## Relationship to the technical specification

Page 27 of `digitmak-portal-v1-technical-spec.pdf` lists exact branding/layout and content approval under “Open Items To Resolve Later”. Therefore V11 provides a complete, reviewable candidate and a formal sign-off checklist; source code alone cannot create institutional approval.
